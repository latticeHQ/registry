# Custom Preset Instructions

This is a placeholder for custom preset instructions. Users should replace this content with their own scenario instructions.

## Template Structure

When creating custom instructions, consider including:

### Character Background
- Name, age, and role
- Personal history relevant to the scenario
- Current situation

### Personality and Behavior
- How the character typically acts
- Communication style
- Key phrases or mannerisms

### Scenario Details
- What brought the character to this situation
- Key concerns or problems
- What the character wants from the interaction

### Learning Objectives
- What skills should the trainee practice
- What outcomes indicate success

### Interaction Guidelines
- How to respond to different approaches
- When to reveal certain information
- Appropriate resistance or cooperation levels
