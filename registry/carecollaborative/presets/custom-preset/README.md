# Custom Preset

This is a template preset that allows users to create their own custom training scenarios.

## How to Use

1. Clone this preset to create your own
2. Customize the metadata in the front matter above
3. Write your own instructions in the instructions.md file
4. Define your own learning objectives

## Configuration Options

- **domain**: Set to "clinical" for healthcare training or "workplace" for HR/workplace training
- **category**: Define your own category or use existing ones
- **sidecar_id**: Use "clinical-v1" for clinical scenarios or "workplace-v1" for workplace scenarios
- **difficulty**: beginner, intermediate, or advanced
- **duration**: Estimated session length

## Learning Objectives

Define your own learning objectives based on your training needs.

## Session Guidelines

Write your own guidelines for how the session should be conducted.
